# expense-tracker# Automated Financial Expense Tracker

This project processes personal financial transactions, categorizes them automatically, and generates basic analytical reports and charts.

## Tech Stack
- Python
- Pandas
- Matplotlib

## Features
✔ Load CSV expense data  
✔ Clean & preprocess  
✔ Auto-categorize using keywords  
✔ Monthly summary  
✔ Visualizations  

## Run

pip install -r requirements.txt
python src/main.py
